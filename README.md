# Fyxer AI Desktop App - Releases

Public repository for distributing Fyxer AI Desktop App releases.

## About

This repository contains compiled releases and auto-update manifests for the Fyxer AI Desktop App. The source code is maintained in a private repository.

## Download

Latest releases are available in the [Releases](https://github.com/Fyxer-AI/desktop-app-releases/releases) section.

### Supported Platforms

- **macOS**: Apple Silicon (ARM64) - `.zip`
- **Windows**: x64 - `.exe` installer and `.zip`

## Auto-Updates

The desktop app automatically checks for updates every 24 hours. When an update is available, you'll receive a notification and can download it in the background.

## Security

All macOS releases are:

- Code signed with a Developer ID certificate
- Notarized by Apple
- Verified by Gatekeeper on first launch
